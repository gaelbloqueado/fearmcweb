Cómo Unirse

Sigue estos sencillos pasos para unirte a nuestra comunidad de FearMC.

1

Asegúrate de tener Minecraft

Necesitas un Cliente de Minecraft Java Edition / Bedrock Edition para poder conectarte a nuestro servidor.

  • Versión compatible: 1.20.x
  • En versiones mas baja podria haber errores.
2

Abre Minecraft y haz clic en "Multiplayer"

En el menú principal de Minecraft, selecciona la opción "Multiplayer" para acceder a la lista de servidores.

Opcion Multiplayer
3

Agrega el servidor FearMC

Haz clic en "Add Server" y completa la información con los siguientes datos:

  • Server Name: FearMC (o el nombre que prefieras)
  • Server Address: fearmc.xyz

En Java Edition no es necesario añadir el puerto, puerto para Bedrock Edition (26542).

Opcion Add Server
4

Conéctate y disfruta

Selecciona el servidor FearMC en tu lista y haz clic en "Join Server". ¡Bienvenido a nuestra comunidad!

Al conectarte por primera vez, te recomendamos leer las reglas del servidor y visitar el spawn para orientarte.

Unirse al servidor FearMC

¿Necesitas ayuda?

Si tienes problemas para conectarte o alguna pregunta sobre el servidor, únete a nuestro servidor de Discord donde nuestro equipo de soporte estará encantado de ayudarte.

Unirse a Discord
